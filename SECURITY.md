# Security Policy

## Supported Versions
We only support two major versions of Bit.
Older versions will still receive security updates however.

| Version | Supported          | Security Updates?          |
| ------- | ------------------ | ------------------ |
| 2025.2.x   | :white_check_mark: | :white_check_mark: |
| 2025.1.x   | :x: | :white_check_mark: |
| 2024.2.x   | :white_check_mark: | :white_check_mark: |
| 2024.1.x   | :x: | :white_check_mark: |
| 5.2.x and below   | :x: | :x: |
